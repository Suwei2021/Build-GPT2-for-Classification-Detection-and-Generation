# Stanford CS224N Default Project
This is the default final project for the Stanford CS224N class. Please refer to the project handout on the course website for detailed instructions. 
## Project Overview
The project consists of three main tasks:
- Sentiment Analysis: Classify the sentiment of a text as positive, negative, or neutral.
- Paraphrase Detection: Identify whether two phrases convey the same meaning.
- Sonnet Generation: Automatically generate textual content.
